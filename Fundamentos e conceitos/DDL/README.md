# DDL - Data Definition Language (Linguagem de Definição de Dados)

Data Definition Language (DDL) é um subconjunto de comandos na linguagem SQL (Structured Query Language) usado para definir e modificar a estrutura de objetos de banco de dados, como tabelas, índices, e esquemas. Os comandos DDL são usados principalmente por administradores de banco de dados e desenvolvedores para criar e estruturar as bases de dados e seus componentes.

## CREATE 

Comando utilizado para criar novos objetos no banco de dados.

## ALTER

Comando para alterar algum objeto no banco de dados.

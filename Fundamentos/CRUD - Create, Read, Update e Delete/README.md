# CRUD

CRUD é um acrônimo para as quatro operações básicas utilizadas em bases de dados relacionais e em qualquer interface que permita a manipulação de dados. O termo CRUD significa:

## CREATE
- Seria a capacidade de criar novos registros. Em termos de SQL, isso geralmente é feito usando o comando INSERT.

## READ
- Seria a capacidade de ler ou recuperar informações existentes. Em SQL, isso é comumente realizado com o comando SELECT.

## UPDATE
- Seria a capacidade de modificar os dados existentes. No SQL, essa operação é executada com o comando UPDATE.

## DELETE
- Seria a capacidade de remover dados. Em SQL, essa operação é realizada com o comando DELETE.

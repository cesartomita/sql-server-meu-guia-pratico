# System stored procedures (Procedimentos armazenados do sistema)

No SQL Server, muitas atividades administrativas e informativas podem ser executadas usando procedimentos armazenados do sistema.

Separei as stored procedures mais utilizadas no dia a dia.

## Database Engine (Mecanismo de Banco de Dados)

- > Fornece informações sobre usuários, sessões e processos atuais em uma instância: [sp_who ou sp_who2](./Database%20Engine/sp_who%20ou%20sp_who2.sql).

- > Retorna informações dos bancos de dados: [sp_helpdb](./Database%20Engine/sp_helpdb.sql).
